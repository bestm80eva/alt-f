Host Setup Help

  • Host name: the box name, as seen by other computers in the network
  • Host description: a more meaningfull box description
  • Workgroup: the MS-W workgroup or the local network domain name
  • Static IP or DHCP:
    Every computer in a network must have an assigned IP. An IP has the form x.x.x.x, where x lies between 1 and 254.
    The IP can be either assigned statically, remaining fixed once for all, or dynamically, where an existing DHCP server in the network leases an IP to the box for a given amount of time, and in this case the IP can change over time. Usually the DHCP server lies in the modem/router that connects you to the internet.
    The recommended setting for computers that are servers, and the DNS-323 is a server, is to have a fixed IP.
    If you select a static IP you have also to specify other network details. If you don't know them, you can start with DHCP, note the values that the DHCP server published, and then revert to a static IP, keeping all DHCP settings but the host IP.
  • Host IP: the IP must be compatible with your other hosts IPs
  • Netmask: how "wide" your local network is, usually 255.255.255.0, meaning that your network can have up to 255 computers
  • Gateway: the IP of the computer that connects you to the internet, generally you modem/router
  • Name server 1: the IP of a computer that translates host names to IP, generally you modem/router or your ISP DNS
  • Name server 2: the IP of a fallback computer that translates host names to IPs, optional.
  • Frame size: Usually 1500. A higher value means that "jumbo frames" are enabled, allowing higher network throughput.
    Maximum value is 9000, but it is a waste of memory, either 3700 (maximum memory saving) or 7500 are good choices. Other computers, routers or switches in the network must also support jumbo frames
  • Enable IPv6: protocol for the next generation internet.
    Most users don't need it. If you activate it it will became immediately active; if you disable it it will become disabled only after the next reboot.
    The kernel-modules package must be installed first.
