Disk Utilities Help

In this page all found disks, including USB disks or pens, will be presented, one per line:
  • Bay will show you where the disk is, in the left or right box bay or on the USB port.
  • Dev. the linux device name
  • Capacity as reported by the disk
  • Disk Model as reported by the disk
  • Eject/Load: this button can be used with USB or internal disks. When
    • Eject appears, it means that some disk resource are in use by the box, and when pressing the button an attempt will be made to stop using them. If the disk is in use by some program or service and it can't be stopped, you have to stop the service or program that is using it before.
    • Load appears, it means that no disk resource is in use, and you can safely remove or eject the disk. If you press the button, all filesystems found on the disk will be mounted and ready to be used. When a disk is plugged-in, this happens automatically.
  • Health: you can see the disk S.M.A.R.T. status or perform some SMART tests.
    You can use Services->System->smart->Configure to automate tests.
    • Show Status shows the disk SMART report, including disk capabilities, SMART disk attributes, and test results.
    • Start short test: as the name says, selecting this performs a fast test, that will run offline (you can continue using the disk). When it finishes you can see the test result selecting Show Status and looking near the top of the SMART Self-test log structure
    • Start long test: this is similar to the short test but is more exhaustive and takes a longer time to accomplish.
  • Power Mode can show StandbyNow to attempt putting the disk in standby, not spinning, or WakeupNow to take the disk out of standby. If some programs or services are using the disk, it can't be put in standby.
  • Power Sav. if the disk supports A.P.M (Advanced Power Management), three levels of power saving can be set. If grayed out, the disk does not supported APM.
    • Low: Low power saving, higher performance, can't spindown
    • Med: Medium power savings and performance, can spindow
    • High: Higher power savings, lower performance, can spindow
    • Disable: disables APM
  • Spindow You can enter the number of minutes of disk inactivity after which it will enter standby mode, stopping spinning. If services are using the disk, it might not be possible to spin it down. This depends on how the service uses the disk. Also, when packages are installed on disk and are running, the disk where the package was installed might not be put in standby; again, it depends on the service that the package offers. You have to press the Submit button for this to take efect.
Swapping on USB devices: Enable if you want to enable swapping on USB devices.
By default swaping is not enabled on USB pens or disks, as it will degrade performance and can shorten the life of your USB pen (that has a limited number of write cycles).
Swap is necessary and is enabled for internal disks, that must have a swap partition as its first partition.
